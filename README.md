Updater 是一个为游戏或程序所设计的专业化、自动化更新客户端。它旨在为用户提供无缝、安全、一键式的模组更新体验，可免去手动下载和覆盖补丁的繁琐步骤。<br>
一键更新：运行程序即可自动完成从检查到安装的全流程，无需任何技术操作。<br>
安全可靠：内置完善的错误恢复机制与磁盘空间检查，更新失败可自动回滚，确保游戏环境不被破坏。<br>
增量更新：支持从任意旧版本升级到最新版本，自动下载并安装所有必需的中间增量包，高效节省流量。<br>
智能清理：根据更新包内的清单，自动清理作废的旧文件，保持游戏目录整洁。<br>
可视进度：集成 tqdm 库，在命令行中显示准确的下载进度条，状态一目了然。<br>
网络优化：具备超时重试机制，应对不稳定的网络环境。<br>
